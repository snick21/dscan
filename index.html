Tranco API

DEVScan uses the Tranco ranking to check if a domain is popular and likely trustworthy. The server looks up the domain in the Tranco list (via local CSV) and uses this as a feature in threat detection.

Isolation Forest

An anomaly detection model in DEVScan’s ML pipeline. It analyzes link features to flag suspicious (anomalous) links.

LightGBM

A machine learning model used in DEVScan to classify links as safe or malicious based on extracted features, improving detection accuracy.

Frequently Asked Questions

Q: What Is DEVScan?

A: DEVScan Is A Chrome Extension That Protects Users From Malicious And Suspicious Links In Real-Time Using Machine Learning And External Threat Intelligence.

Q: How Does DEVScan Warn Me About A Risky Link?

A: DEVScan Highlights Links In Three Colors:

  • 🔴 Red For Confirmed Threats
  • 🟡 Yellow For Anomalies
  • 🟢 Green For Safe Links

Q: Will DEVScan Slow Down My Browser?

A: No. DEVScan Is Designed To Run Efficiently In The Background And Only Scans Links, So Your Browsing Stays Smooth.

Q: Does DEVScan Collect My Data?

A: No. We Prioritize Privacy And Do Not Store Personal Information Or Browsing History. All Threat Detection Happens Securely.

Q: Can I Disable Auto-Blocking?

A: Yes. Auto-Blocking Is Optional. You Can Enable Or Disable It In The Extension Settings.

Q: Is There A Mobile Version?

A: Currently, DEVScan Is Only Available For Desktop Chrome Browsers. Mobile Support Is Planned In Future Updates.
